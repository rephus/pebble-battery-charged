Get notified when you charge your pebble and the battery level reaches 80%. This way, you can manually interrupt the charge. Preventing batteries to charge completely has been proven to improve their capacity degradation.

When 80% charge is reached, the pebble will display a message and vibrate repeatedly. Additionally, you can configure your pushbullet access token or a custom webhook in the settings so that you get a remote notification as well.

Install from the pebble app store: 

https://apps.repebble.com/e7540c9775d545af81ba7f3f
